# DuSwToglTF

## 介绍(Introduction)

A SolidWorks Addin to save SolidWorks file as glTF formats --Solidworks插件用来将SoldiWorks文件转换为glTF格式。将sldprt和sldasm转换为gltf。

1. Support *.gltf *.glb;
2. Support PartDoc and AssemblyDoc;
